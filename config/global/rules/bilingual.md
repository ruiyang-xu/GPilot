# Bilingual (EN/CN) Rules

## Translation Workflow

1. Always draft in English first
2. Translate to Chinese as a second pass
3. Chinese version is NOT a literal translation — adapt for Chinese reader context

## Chinese Style

- Use professional financial Chinese (财经中文)
- Avoid overly academic or bureaucratic tone
- Target audience: Chinese tech/VC community, WeChat readers
- Use standard Chinese financial terms: 估值, 融资, 独角兽, 赛道, 护城河
- Keep English terms for widely-used ones: SaaS, AI, IPO, SPAC, PE/VC

## English Style

- Professional but engaging, similar to top-tier research houses
- Target audience: LinkedIn professional network, global investors
- Avoid jargon without explanation for non-specialist readers

## Publication Format

- WeChat 公众号: Chinese version, formatted for mobile reading
- LinkedIn: English version, formatted for professional network
- Both versions should be standalone — reader should not need the other language version

## Names & Terms

- Company names: use English name, add Chinese in parentheses on first mention if well-known
- Technical terms: keep English if no standard Chinese equivalent exists
- Financial metrics: keep in original currency, convert to USD if needed
