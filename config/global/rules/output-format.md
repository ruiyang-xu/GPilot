# Output Format Preferences

## General

- Lead with conclusion/recommendation, then supporting analysis
- Use bullet points for lists, tables for structured comparisons
- Keep paragraphs short (3-4 sentences max)
- Use headers to organize long documents

## Financial Data

- Currency: USD unless specified otherwise
- Large numbers: $50M, $1.2B (not $50,000,000)
- Multiples: 2.5x, 10x (not 2.5 times)
- Returns: 15% IRR, 3.2x MOIC
- Dates: YYYY-MM-DD for data, natural language for prose

## Research Publications

- Target length: Market Note 1,500-2,000 words; Company Teardown 3,000-5,000 words
- Always include: key takeaway box at top, data tables, source citations
- Charts: describe data for visualization agent to render
- Tone: authoritative but accessible, not academic

## Documents

- Investment memos: structured format per Fund/templates/investment-memo.md
- LP reports: follow ILPA quarterly report template
- Emails: concise, action-oriented, professional

## Code & Technical

- Prefer Python for data analysis scripts
- Use markdown for documentation
- Comment sparingly — code should be self-explanatory
