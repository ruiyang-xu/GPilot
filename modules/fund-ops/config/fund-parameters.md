# Fund Parameters

> Configure these to match your fund's investment strategy.

## Check Sizes

| Stage | Check Size | Ownership | Follow-on |
|-------|-----------|-----------|-----------|
| Pre-seed/Seed | $500K–$2M | 5–10% | 2x |
| Series A | $2M–$5M | 5–10% | 1.5x |
| Series B | $5M–$10M | 3–7% | 1x |
| Growth | $10M–$20M | 1–5% | Case-by-case |

## Fund Terms

Terms: 2% mgmt fee | 20% carry over 8% hurdle | European waterfall | 10yr + 2x 1yr extensions

## Usage

These parameters are used by:
- `/deal-screen` — check size vs. fund capacity
- `/ic-memo` — ownership and follow-on calculations
- `/portfolio-review` — return analysis and reserve tracking
- `skills/fund-accounting/` — fee and waterfall calculations

To enable, copy this file to `config/fund-parameters.md` or reference from CLAUDE.md.
